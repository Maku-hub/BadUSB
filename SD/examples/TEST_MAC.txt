REM Opens TextEdit via Spotlight and exercises every supported command,
REM character and key combination. Observable tests use an "EXPECT:" line.
REM Assumes US keyboard layout.
REM
REM macOS differences vs Windows/Linux, reflected below:
REM   - Clipboard uses GUI (Cmd): GUI c / GUI v / GUI x  (not CTRL).
REM   - Line start/end is GUI LEFT / GUI RIGHT (Cmd+Left/Right), not HOME/END.
REM   - Word selection is ALT SHIFT LEFT (Option+Shift+Left), not CTRL SHIFT.
REM   - No F5 date insert, no Menu/App/Insert/Break keys.
REM TIP: turn OFF TextEdit smart quotes (Edit > Substitutions) or the ' and "
REM in section [A] will appear as curly quotes - that is TextEdit, not the device.
REM EXCLUDED (would abort the test): GUI q (quit), GUI w (close window),
REM CTRL + arrows (switches Spaces).
REM ================================================================
